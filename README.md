# PaperAgent

PaperAgent 是一个面向论文阅读与学术汇报的智能分析 Agent。

## 项目目标

本项目计划实现一个可以处理 PDF / Markdown / TXT 文档的智能体系统，支持：

1. 文档解析 √
2. 文档检索 √
3. 论文总结 √
4. 方法分析 √
5. 实验分析 √
6. Concern 生成
7. PPT 大纲生成
8. Markdown 报告导出

..... TBC

## 技术栈

- Python
- VSCode
- Git / GitHub
- Streamlit
- PyMuPDF
- Chroma / FAISS
- LLM API
- RAG
- Agent Loop
